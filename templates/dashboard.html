Deploy to Railway (Free — Always On)
1
Push code to GitHub
Create a GitHub repo and push all files from the jobblast_hosted/ folder. Make sure recruiters.csv and resume.pdf are included.
2
Create free Railway account
Go to railway.app → Sign up with GitHub → New Project → Deploy from GitHub repo. Select your jobblast repo.
3
Set environment variables
In Railway → Variables tab, add:
GMAIL_USER = venkatachetanofficial@gmail.com
GMAIL_PASS = your app password
PORT = 5000
4
Deploy & get your live URL
Railway auto-deploys and gives you a URL like jobblast-xyz.railway.app. Open that URL to see this dashboard. Your automation runs 24/7 even when your PC is off.
5
Done! ✅
Emails go out every morning at 8:00 AM IST automatically. Use the ▶ Run Now button above to trigger manually anytime.
